# Seller E-commerce (React Learning Project)

Yeh project React revise karne ke liye bhi ready hai, aur codebase me Hindi (Roman) comments add kiye gaye hain taaki flow jaldi samajh aaye.

## Run project
```bash
npm install
npm run dev
```

## Build project
```bash
npm run build
npm run preview
```
